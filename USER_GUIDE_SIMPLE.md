# 🎯 用户指南 - 超简单 Agent 配置

## ⚡ 1 分钟快速开始

### 第 1 步：打开直播间
访问：http://localhost:3000/rooms/test

### 第 2 步：点击设置
点击右上角的 **⚙️ 设置** 按钮

### 第 3 步：配置 Agent

#### 如果想快速测试（推荐新手）
1. 选择 **"演示模式"**
2. 勾选 **"启用 Agent"**
3. 点击 **"保存设置"**
4. 完成！

#### 如果想连接真实 OpenClaw Agent
1. 选择 **"真实 Agent（OpenClaw）"**
2. 勾选 **"启用 Agent"**
3. 粘贴你的 **Bot Token**（从 Telegram @BotFather 获取）
4. 点击 **"自动获取 Chat ID"**（需要先给 Bot 发过消息）
5. 点击 **"测试连接"** 确认成功
6. 点击 **"保存设置"**
7. 完成！

## 🔧 获取 Telegram 信息

### Bot Token
1. Telegram 搜索 `@BotFather`
2. 发送 `/mybots`
3. 选择你的 Bot
4. 点击 "API Token"
5. 复制整个 Token

### Chat ID
**方法 1（推荐）：**
1. 给你的 Bot 发送任意消息（例如：`/start`）
2. 在设置面板点击 **"自动获取 Chat ID"**
3. 自动填充完成！

**方法 2（手动）：**
1. 给你的 Bot 发送 `/start`
2. 浏览器访问（替换 `<Token>` 为你的 Bot Token）：
   ```
   https://api.telegram.org/bot<Token>/getUpdates
   ```
3. 找到 `"chat":{"id":数字}`
4. 复制这个数字

## ❓ 常见问题

### Q: "自动获取"按钮无法点击
**A:** 请先填写 Bot Token

### Q: 自动获取显示"没有找到消息"
**A:** 请先在 Telegram 给你的 Bot 发送一条消息

### Q: 测试连接失败
**A:** 检查：
- Bot Token 是否正确（完整复制）
- Chat ID 是否正确
- 是否给 Bot 发过消息
- 网络连接是否正常

### Q: Agent 不响应
**A:** 检查：
- 是否已"保存设置"
- 是否已"开始直播"
- OpenClaw Bot 是否在线
- 查看浏览器控制台有无错误

### Q: 想切换回演示模式
**A:** 打开设置，选择"演示模式"，保存即可
