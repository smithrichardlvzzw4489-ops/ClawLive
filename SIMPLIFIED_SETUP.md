## 📱 现在就试试

### 步骤 1：启动服务（如果还没启动）

**后端**：
```bash
cd apps/server
pnpm dev
```

**前端**：
```bash
cd apps/web
pnpm dev
```

### 步骤 2：访问直播间

打开浏览器：http://localhost:3000/rooms/test

### 步骤 3：配置用户身份

1. 点击右上角 **"⚙️ 设置"**
2. 选择 **"🌟 Telegram 用户身份"**
3. 打开 **"启用 Agent"** 开关
4. 输入你的手机号（如：`+8613800138000`）
5. 输入 Agent Chat ID（如：`@your_agent`）
6. 点击 **"📱 发送验证码"**
7. 打开 Telegram 查看验证码
8. 输入验证码
9. 点击 **"✅ 提交验证码"**
10. 完成！🎉

### 步骤 4：开始直播

直接发消息就行了！你的消息会以**真实用户身份**出现在 Telegram 中！

## 📊 技术细节

### 后端改进
- `startLogin()` - 发送验证码
- `submitCode()` - 提交验证码
- `submitPassword()` - 提交两步验证密码（可选）

### 前端改进
- 分步UI界面（phone → code → password → done）
- 自动状态管理
- 实时错误反馈

### 配置简化

- 无需手动配置环境变量
