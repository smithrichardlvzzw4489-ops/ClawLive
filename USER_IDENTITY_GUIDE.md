# 🌟 用户身份直播指南（超简单版）

## 什么是"用户身份"？

让你在直播间发送的消息，在 Telegram 中显示为**你本人**发的，不是 Bot 发的！

### 效果对比

**之前（Bot）**：
```
Telegram 显示：
  Bot: 🎙️ [主播]: 你好
  Agent: 你好！
```

**现在（你）**：
```
Telegram 显示：
  你: 你好
  Agent: 你好！
```

## 🚀 3 步配置（1 分钟搞定）

### 第 1 步：打开设置

1. 进入直播间：http://localhost:3000/rooms/test
2. 点击右上角"⚙️ 设置"按钮
3. 选择 **"🌟 Telegram 用户身份"**
4. 启用 Agent

### 第 2 步：登录 Telegram

在弹出的界面中：

1. **输入手机号**（带国家码，如：`+8613800138000`）
2. **输入 Agent Chat ID**（你的 Agent 用户名，如：`@your_agent`）
3. 点击 **"📱 发送验证码"**
4. 打开 Telegram 查看验证码
5. **输入验证码**
6. 点击 **"✅ 提交验证码"**

> 💡 如果你的账号有两步验证，会要求输入密码

### 第 3 步：开始直播！

登录成功后，直接发消息就行了！你的每条消息都会以**真实身份**出现在 Telegram 中。

## ✅ 就这么简单！

不需要：
- ❌ 编辑配置文件

只需要：

## 🔒 安全性

- 你的登录凭证安全地保存在服务器内存中
- 只要不重启服务器，就会一直保持登录状态
- 可以随时在设置中切换回 Bot 模式

## ❓ 常见问题

### Q: 需要手机验证码？
A: 是的，这是 Telegram 的安全机制，但只需要验证一次。

### Q: 验证码收不到？
A: 检查手机号格式是否正确（需要带国家码，如 +86）

### Q: 提示需要两步验证密码？
A: 如果你的 Telegram 账号开启了两步验证，需要输入密码。

### Q: 重启服务器后需要重新登录吗？
A: 是的，目前需要重新验证（未来会改进）。
